# 系统信息架构

## 1. 一级导航

### 工作台

用途：全局状态总览和待处理事项入口。

主要内容：
- 运行指标
- 异常告警
- 待发布草稿
- 评测失败项
- MCP 工具异常
- 最近运行记录

### 智能体

用途：管理所有智能体资产。

主要页面：
- 智能体列表
- 智能体详情
- 智能体配置
- 版本记录
- 绑定资源

### 工作流编排

用途：以节点画布方式定义智能体执行流程。

主要区域：
- 节点画布
- 节点库
- 节点属性面板
- 调试控制台
- 运行结果面板

### MCP / 工具

用途：管理外部能力连接。

主要页面：
- MCP Server 列表
- 工具详情
- Schema 查看
- 权限配置
- 调用日志
- 密钥和环境变量

### 知识库

用途：管理可检索知识资源。

主要页面：
- 知识库列表
- 文档管理
- 分段配置
- 检索测试
- 绑定智能体

### 评测

用途：发布前质量检查和效果度量。

主要页面：
- 测试集列表
- 测试用例
- 批量评测运行
- 评测报告
- 发布门禁配置

### 运行日志

用途：运行级观测和问题排查。

主要页面：
- 运行列表
- Trace 详情
- 模型调用
- 工具调用
- 检索命中
- 错误分析

### 发布渠道

用途：将智能体发布到真实用户入口。

主要页面：
- Web Chat
- API
- 企业 IM 渠道
- 发布版本
- 回滚记录

### 设置

用途：平台级配置和治理。

主要页面：
- 空间管理
- 用户与角色
- 模型供应商
- 全局密钥
- 审计日志
- 环境配置

## 2. 核心工作流

### 创建并发布智能体

1. 创建智能体草稿。
2. 配置基础信息、模型和提示词。
3. 进入工作流编排器。
4. 添加知识库检索节点和 MCP 工具节点。
5. 运行调试并查看 trace。
6. 绑定测试集并运行评测。
7. 通过发布检查。
8. 发布到测试环境。
9. 发布到生产渠道。
10. 在运行日志中持续观测。

### 接入 MCP 工具

1. 添加 MCP Server。
2. 配置连接参数和密钥。
3. 同步工具列表和 schema。
4. 配置工具权限。
5. 在工作流工具节点中选择工具。
6. 调试工具调用。
7. 在运行日志中观察调用结果。

### 知识库接入

1. 创建知识库。
2. 导入文档。
3. 配置分段和 Embedding。
4. 运行检索测试。
5. 绑定到智能体或节点。
6. 在 trace 中查看引用和命中片段。

## 3. 页面优先级

P0：
- 工作台总览
- 智能体列表
- 智能体详情
- 工作流编排器
- MCP 工具管理
- 运行 Trace 详情

P1：
- 知识库管理
- 评测页面
- 发布配置
- 权限设置

P2：
- 审计日志
- 高级模型路由
- 企业 IM 渠道
- 模板市场
