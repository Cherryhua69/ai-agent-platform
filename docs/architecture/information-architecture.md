# 系统信息架构

## 1. 一级导航

### 工作台

用途：企业员工统一使用 Agent 的任务工作台，全局状态总览和待处理事项入口。

主要内容：
- 任务列表区
- 对话/任务执行区
- 文件区
- 日志区
- 工具面板
- Agent 配置入口
- 权限与安全入口
- 运行指标
- 异常告警
- 待发布草稿
- 评测失败项
- MCP 工具异常
- 最近运行记录

### 智能体

用途：管理所有智能体资产。

主要页面：
- 智能体列表
- 智能体详情
- 智能体配置
- 版本记录
- 绑定资源
- 安全配置
- API 发布配置
- 观测配置

### 工作流编排

用途：以节点画布方式定义智能体执行流程。

主要区域：
- 节点画布
- 节点库
- 节点属性面板
- 调试控制台
- 运行结果面板

节点能力：
- LLM
- 条件分支
- 知识库检索
- MCP / API 工具调用
- 数据库查询
- 代码执行
- 文件读写
- Agent 协作
- 人工转接

### MCP / 工具

用途：管理外部能力连接。

主要页面：
- MCP Server 列表
- 工具详情
- Schema 查看
- 权限配置
- 调用日志
- 密钥和环境变量
- API 工具管理

### 数据库连接

用途：管理 Agent 可使用的数据库连接和查询权限。

主要页面：
- 数据库连接列表
- 连接详情
- 访问权限
- 只读/可写策略
- 查询日志
- 高风险确认规则

### 知识库

用途：作为平台级核心模块统一管理可检索知识资源、文档和向量索引。

主要页面：
- 知识库列表
- 知识库详情
- 文档管理
- 向量索引管理
- 分段配置
- 检索测试
- 绑定智能体
- 权限范围
- 使用统计

核心视图：
- 知识库列表：展示所有知识库、分类、负责人、状态、文档数量、索引状态、被哪些 Agent 使用。
- 知识库详情：展示基础信息、权限范围、绑定 Agent、最近更新时间、使用统计。
- 文档管理：支持上传、删除、重新解析、重新向量化、查看处理状态。
- 检索测试：管理员可以输入问题，测试该知识库的召回结果、相似度、来源片段和排序效果。

### 评测

用途：发布前质量检查和效果度量。

主要页面：
- 测试集列表
- 测试用例
- 批量评测运行
- 评测报告
- 发布门禁配置

### 运行日志

用途：运行级观测和问题排查。

主要页面：
- 运行列表
- Trace 详情
- 模型调用
- 工具调用
- 检索命中
- 数据库查询
- 代码执行
- 文件操作
- Agent 协作
- 错误分析

### API 发布管理

用途：将智能体发布为 API 或其他真实用户入口，供第三方系统调用。

主要页面：
- API
- Web Chat
- 企业 IM 渠道
- 发布版本
- 回滚记录
- 认证方式
- 调用方白名单
- 请求/响应 Schema
- 并发与日志保留

### 设置

用途：平台级配置和治理。

主要页面：
- 空间管理
- 用户与角色
- 模型供应商
- 全局密钥
- 审计日志
- 环境配置
- 高风险操作确认规则
- 文件访问策略
- API 调用策略

## 2. 核心工作流

### 创建并发布智能体

1. 创建智能体草稿。
2. 配置基础信息、模型和提示词。
3. 进入工作流编排器。
4. 添加知识库检索节点、MCP/API 工具节点、数据库查询节点或代码/文件节点。
5. 运行调试并查看 trace。
6. 绑定测试集并运行评测。
7. 通过发布检查。
8. 发布到测试环境。
9. 发布为 API 或生产渠道。
10. 在运行日志中持续观测。

### 接入 MCP 工具

1. 添加 MCP Server。
2. 配置连接参数和密钥。
3. 同步工具列表和 schema。
4. 配置工具权限。
5. 在工作流工具节点中选择工具。
6. 调试工具调用。
7. 在运行日志中观察调用结果。

### 知识库接入

1. 创建知识库。
2. 设置分类、负责人、可见范围和状态。
3. 导入文档。
4. 执行文件解析、内容清洗、文档切分、向量化、索引构建和入库。
5. 查看文档处理状态、索引状态和错误信息。
6. 运行检索测试。
7. 绑定到智能体或节点。
8. 在 trace 中查看引用和命中片段。

### 发布 Agent API

1. 选择待发布 Agent 版本。
2. 配置 API 路径、认证方式、调用方白名单、请求参数 Schema 和响应格式 Schema。
3. 执行发布检查，确认评测结果、知识库权限、工具状态、数据库权限和高风险能力。
4. 发布到测试环境或生产环境。
5. 第三方系统调用 API。
6. 在任务与运行日志中查看调用方、请求、响应、状态、耗时和错误。

- 智能体列表
- 智能体详情
- 工作流编排器
- MCP 工具管理
- 知识库管理
- 文档管理
- 向量索引管理
- API 发布管理
- 任务与执行日志
- 运行 Trace 详情

P1：
- 评测页面
- 数据库连接管理
- API 工具管理
- 权限设置
- 审计日志

P2：
- 高级模型路由
- 企业 IM 渠道
- 模板市场
- 多 Agent 协作治理
