# 工作流节点交互优化设计

## 目标

简化输出节点与注释节点的画布交互，并支持通过键盘删除节点连接线。

## 输出节点

- 输出节点使用 `FileOutput` 图标，与用户输入和 LLM 节点保持一致的标题图标结构。
- 节点卡片只显示节点标题，不显示已配置的输出变量，也不显示“请配置输出变量”占位内容。
- 输出变量仍在右侧详细配置中编辑和保存。

## 注释节点

- 单击注释节点仅选中节点，不打开右侧详细配置面板。
- 双击注释节点后，卡片内容原地切换为多行文本框并自动聚焦。
- 文本变化时实时更新节点描述并触发工作流自动保存，不需要确认按钮。
- 文本框失焦后退出编辑状态，保留最新内容。
- 注释节点仍可移动和删除，位置保存逻辑保持不变。

## 连接线删除

- 鼠标点击连接线后使用 React Flow 原生选中状态高亮连接线。
- 按键盘 `Delete` 或 `Backspace` 删除选中的连接线。
- 删除操作通过现有 `onEdgesChange` 更新画布状态并触发自动保存。
- 未选中连接线时按删除键不影响节点或其他画布内容。

## 测试

- 输出节点卡片存在 `FileOutput` 图标且不渲染输出变量摘要。
- 单击注释节点不出现节点配置面板，双击后出现文本框，输入会实时保存。
- 选中连接线并按 `Delete` 或 `Backspace` 后连接线消失，保存请求不再包含该连接线。
